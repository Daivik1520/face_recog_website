# 🧠 FRAMS AI — Face Recognition Web Interface

**A sleek, AI-powered face recognition landing page with smooth animations, 3D parallax, and a modern glassmorphism UI.**

[🌐 Live Demo](https://frams-ai.netlify.app/) | [📁 Source Code](https://github.com/Daivik1520/face_recog_website)

</div>

---

## ✨ Features

- 🎭 **Smooth Scroll** — Powered by Lenis for a buttery-smooth scrolling experience  
- 🪟 **Glassmorphism UI** — Modern frosted-glass design aesthetic  
- 🔄 **3D Parallax Mockup** — Mouse-tracked 3D rotation on the hero section  
- 🔤 **Alphabet Reveal Animation** — Characters animate on scroll  
- 📊 **Animated Stat Counters** — Numbers count up dynamically  
- 📱 **Fully Responsive** — Mobile-friendly with smooth navigation  
- 🔦 **Scroll Reveal** — Sections animate into view using IntersectionObserver

## 🚀 Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/Daivik1520/face_recog_website.git
cd face_recog_website/FACE\ RECOG\ WEB
```

### 2. Open in browser
```bash
# Open the file directly
open index.html
```

> No build tools or dependencies required — pure frontend project.
