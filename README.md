# 🧠 FRAMS AI — Face Recognition Web Interface

**A sleek, AI-powered face recognition landing page with smooth animations, 3D parallax, and a modern glassmorphism UI.**

[🌐 Live Demo](https://frams-ai.netlify.app/) &nbsp;|&nbsp; [📁 Source Code](https://github.com/Daivik1520/face_recog_website)

</div>

---

## ✨ Features

- 🎭 **Smooth Scroll** — Powered by [Lenis](https://github.com/studio-freight/lenis) for buttery-smooth scrolling experience
- 🪟 **Glassmorphism UI** — Modern frosted-glass design aesthetic throughout
- 🔄 **3D Parallax Mockup** — Mouse-tracked 3D rotation on the hero glass mockup
- 🔤 **Alphabet Reveal Animation** — Characters animate in as you scroll
- 📊 **Animated Stat Counters** — Metrics count up dynamically on scroll into view
- 📱 **Fully Responsive** — Mobile-ready with a burger menu and smooth nav transitions
- 🔦 **Scroll Reveal** — Sections fade/slide in with staggered timing via IntersectionObserver

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/Daivik1520/face_recog_website.git
   cd face_recog_website/FACE\ RECOG\ WEB
   ```

2. **Open in browser**
   ```bash
   # Simply open index.html in any modern browser
   open index.html
   ```

> No build tools or dependencies required — pure frontend project.
